## Project summary
OLOVA is a skater-owned company rooted in the Tri-Cities of British Columbia. The site is built around:
1. selective products
2. design and visualization
3. community-linked work

The goal is a public-facing website that feels real, local, and serious, without sounding corporate, vague, or over-claimed.

## Biggest lessons learned
### Copy
- The site easily drifts into internal notes-to-self language.
- The fix is to write publicly first, then refine.
- Strongest tone is:
  - skater-owned
  - Tri-Cities rooted
  - selective
  - clear
  - measured
- Weakest tone is:
  - meta
  - consultant-like
  - architecture-firm cosplay
  - forced slang
  - lane language used as site self-explanation

### Technical workflow
- GitHub app is safe for text commits when using plain UTF-8.
- GitHub app binary blob workflow corrupted media files.
- Safe rule:
  - text through GitHub app
  - images and video through direct GitHub web upload unless individually verified
- Google Drive is the archive and source library.
- Chat uploads are the best way to shortlist and judge images.
- Asset pack workflow works well:
  - assistant selects
  - assistant optimizes and names files
  - user uploads asset pack to repo if needed
